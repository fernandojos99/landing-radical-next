## Descargar 

```
git clone https://github.com/fernandojos99/landing-radical-next.git 
```


## Instalar dependencias

```
npm install
```

## Variables de entorno 

se requieren las siguientes variables de entorno 

```
DATABASE_URL=<la pooler>
SUPABASE_URL=
# La de abajo no se usa
#SUPABASE_ANON_KEY=
SUPABASE_SERVICE_ROLE_KEY=
# Se obtienen de proyecto abacus
ABACUSAI_API_KEY=
WEB_APP_ID=
NOTIF_ID_NEW_REGISTRATION_SUBMISSION=

```


## Ejecutar proyecto 

```
npm run dev
```
