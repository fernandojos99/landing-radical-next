# Radical Innovation 2026 - Landing Page

## Overview
Bilingual (ES/EN) landing page for "Radical Innovation" — a Shark Tank-style innovation competition organized by Grupo Salinas × Universidad de la Libertad × IRIS StartUp Lab.

## Technical Stack
- Dark theme by default (force dark, no system theme)
- Bilingual via React Context (`LocaleProvider`) — Spanish primary, English secondary
- Framer Motion animations throughout
- Prisma ORM for registration data storage
- S3 cloud storage for file uploads
- Email notifications via Abacus AI notification API

## Visual Style
- **Theme**: Dark background (#0a0e1a), cyan primary (#22d3ee), purple secondary accents
- **Typography**: Plus Jakarta Sans for display, DM Sans for body, JetBrains Mono for monospace
- **Design**: Gradient text accents (cyan→purple), glow borders, card-based layouts with subtle borders
- **Animations**: FadeIn, SlideIn, Stagger from built-in animate components + framer-motion

## Pages
1. **`/`** — Landing page with sections: Hero, About, Profiles, Requirements, Process (timeline), Criteria, Dates, Incentives, CTA, Footer
2. **`/register`** — Multi-step registration form (6 steps) with file upload
3. **`/admin`** — Password-protected admin panel (password: RadicalAdmin2026) with data table, search, filter, CSV export

## Key Implementation Details
- **Logos**: `/assets/iris-logo.png` and `/assets/ul-logo.png` in public folder
- **Admin Password**: RadicalAdmin2026 (simple query param auth, not NextAuth)
- **Email**: Notifications sent to amayaandres95@gmail.com on new registration
- **Notification ID**: NOTIF_ID_NEW_REGISTRATION_SUBMISSION env var
- **Database**: Single `Registration` model with all form fields
- **File Upload**: Presigned URL upload to S3 (private files)
- **i18n**: Translations in `lib/i18n.ts`, context in `lib/locale-context.tsx`
